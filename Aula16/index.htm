Agrupamento de Seletores

podemos aplicar as mesmas regras CSS para vários seletores diferentes. para isso, poedemos separar os seletores ultilizando uma vírgula.

Assim não precisasmos ficar copiando e colando colandos os mesmos códigos para todos, nós podemos fazer um só bloco CSS e reutizalizar as mesmas regras!
