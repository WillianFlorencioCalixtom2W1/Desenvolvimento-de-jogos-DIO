Cores

Existem várias formas de definirmos cores cores ultilizando o CSS. Elas são:

  • Cores pré-definidas.
  • Palavra-chave "Currentcolor".
  • Hexadecimal.
  • Hexadecimal com transparencia.
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl(hue, saturation, ligthness)

  • Hue (matiz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Ligthness luminosidade: luminosidade da cor
    • 0% é preto
    • 100% é branco
