No CSS nós possuímos os seletores por ID

  • Seletor por Tipo: Busca um elemento por tag HTML específica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe(.):Busca elementos através do atributo "."
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributoa (latril): Seleciona todos os elementos que possuem um atributo específico em nosso documento HTML e também conseguimos buscar atributos com valor específico.
